# To-Do List Python Project

This is a simple To-Do List application built using Python.  
It allows users to manage their daily tasks efficiently.

🔗 Run the project here:  
https://www.online-python.com/KlFs8Zwhqu

## Features
- Add new tasks
- View existing tasks
- Delete tasks
- Save tasks to a file
- Load saved tasks automatically

## How it works
The program uses file handling to store tasks in a text file and retrieves them when the program starts. It provides a simple menu-based interface for user interaction.

- Python
